JQSelect - jQuery 下拉框插件

描述

依赖于jQuery库实现的下拉框插件，实现加载远程数据，在值改动前后，及数据加载完成或请求失败时均可进行额外操作

示例

属性

名称
值类型
默认值

描述

name
类型:String
默认值:null
描述:

控件的name属性，可以在表单提交时直接应用

width
类型:number
默认值:100
描述:

控件的宽度

height
类型:number
默认值:30
描述:

控件的高度

textAlign
类型:String
默认值:center
描述:

选中选项的文字水平对齐方式，可选值：center,left,right

valueField
类型:String
默认值:value
描述:

用于绑定选项基础数据值

textField
类型:String
默认值:text
描述:

用于绑定选项基础数据名称

url
类型:String
默认值:null
描述:

通过url加载远程列表数据

method
类型:String
默认值:get
描述:

请求远程数据的方式，可选值：get,post

queryParams
类型:Object
默认值:{}
描述:

请求远程数据的附加查询条件

data
类型:Array
默认值:null
描述:

本地数据，会覆盖远程请求的数据

filter
类型:Function
默认值:
描述:

用于过滤数据的函数，返回值必须是包含一组对象的数组

事件

名称
参数

描述

onChange
参数:newVal, oldVal
描述:

在选项值改变时触发

onBeforeSelect
参数:record
描述:

在选项选择之前触发

onAfterSelect
参数:record
描述:

在选项选择之后触发

onBeforeLoad
参数:params
描述:

在请求远程数据之前触发，返回false取消请求远程数据

onLoadSuccess
参数:data
描述:

在请求远程数据成功并且加载数据成功时触发

onLoadError
参数:error
描述:

在请求远程数据失败时触发

方法

名称
参数:参数
描述:

描述

options
参数:none
描述:

返回属性对象

getData
参数:none
描述:

返回加载数据

loadData
参数:data
描述:

加载本地列表数据

reload

根据'url'参数重写原始URL，加载远程列表数据

setValue

描述:

设置下拉框的值

getValue

描述:

clear
参数:none

清空下拉框的值
